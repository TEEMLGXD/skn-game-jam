เกี่ยวกับกิจกรรม

SKN Game Jam เป็นกิจกรรมสำหรับนักเรียนที่สนใจในการเขียนโปรแกรมและสร้างเกม
ไม่ว่าจะเป็นมือใหม่ที่ไม่เคยสร้างเกมมาก่อน หรือผู้ที่มีประสบการณ์ทำเกมมาก่อนก็เข้าร่วมได้!
โดยในกิจกรรมน้องๆจะได้อบรมตั้งแต่การออกแบบเกมและเลือก Game Engine ที่เหมาะกับน้องๆ และจะมีคลาสสอนสำหรับแต่ละ Game Engine ที่น้องๆเลือก

หลังจากอบรมเสร็จ เราจะแข่งขันสร้างเกมภายใน 72 ชม. ตามหัวข้อที่่กำหนด โดยมีพี่ๆคอยช่วยเหลือตลอดเวลา

Unity

เอนจินที่ใช้กันทั่วโลกในอุตสาหกรรมเกม เหมาะสำหรับคนที่อยากทำเกม 2D หรือ 3D จริงจัง มีระบบพร้อมทุกอย่าง ตั้งแต่ฟิสิกส์ แสง เงา ไปจนถึงระบบตัวละคร รองรับทั้งมือถือ,คอม และเว็บ ทำให้เป็นตัวเลือกยอดนิยมของ Game Jam และสามารถต่อยอดได้มาก และหลากหลาย

Gdevelop

ใช้งานง่าย ไม่ต้องเขียนโค็ด เหมาะสำหรับมือใหม่และคนที่ไม่เคยสร้างเกมมาก่อน เหมาะสำหรับ 2D มีระบบ Event-Based ที่ช่วยให้สร้างเกมได้เร็วมาก ภายในไม่กี่ชั่วโมงก็ทำต้นแบบเกมได้แล้ว ส่งลงเว็บเล่นได้ทันที เหมาะสำหรับการสร้างเกม 2D เล็กๆในเวลาที่จำกัด

Roblox Studio

เริ่มง่าย ทำได้เร็ว และสนุก เหมาะสำหรับคนที่อยากสร้างเกม 3D แบบ Multiplayer มี Asset ให้เลือกเป็นร้อย พร้อมระบบฟิสิกส์และผู้เล่นหลายคนแบบสำเร็จรูป ตัวเลือกที่ดีสำหรับคนอยากเริ่มทำเกมครั้งแรกแบบไม่ต้องเริ่มจากศูนย์

กติกาสำหรับ Game Jam

  1. แข่งขันเป็นทีม ทีมละ 2-5 คน
  2. ใช้ Game Engine ใดก็ได้ ทั้งที่มีสอนและไม่มี ตามที่น้องถนัด
  3. สามารถนำผลงานจากเวิร์คช็อปมาพัฒนาต่อในการแข่งขันได้
  4. สามารถใช้ Asset ภายนอกได้แต่ต้องเป็น Open Source เท่านั้น
  5. สามารถใช้ Ai ได้ไม่ว่าจะเป็น Code หรือ Asset
  6. แต่ละทีมเลือก Diversifiers มากสุด 2 อย่าง โดยแต่ละอันจะช่วยเพิ่มคะแนนได้หากทำตรงโจทย์ที่เลือกมา
  7. ส่งไฟล์เกมตามเวลาที่กำหนด (หากสามารถ Export ได้ทันเวลา)
